本仓库托管了我在学习 JavaScript 的过程中写下的示例代码和练习代码，作为代码笔记以便时常温习。

本仓库中所有的代码笔记实际上都是作为我的 JavaScript 学习笔记的附录进行整理的，它们都有对应的博客文章来记录更详细的笔记内容，可访问我的博客来查看更多笔记内容，欢迎各位交流学习、批评指正。
